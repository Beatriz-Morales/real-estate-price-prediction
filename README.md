# 🏠 Real Estate Price Prediction

**Course:** MBA Business Data Analytics  
**Tools:** R · ggplot2 · dplyr · regression modeling  
**Dataset:** Real estate valuation dataset with transaction, location, and neighborhood features

---

## Business Problem

Can property price per unit area be predicted using location and neighborhood characteristics?  
This project applies regression modeling to estimate real estate value and identify which factors most strongly influence housing prices.

---

## What I Did

**1. Data Preparation**  
- Loaded and reviewed the real estate dataset in R
- Examined variable types, missing values, and summary statistics
- Prepared predictors related to transaction timing, age of house, distance to MRT, convenience stores, latitude, and longitude
- Explored relationships between predictors and price per unit area

**2. Regression Modeling**  
- Built multiple linear regression models to predict house price per unit area
- Evaluated model fit and coefficient significance
- Compared variables to identify the strongest pricing drivers
- Interpreted the direction and magnitude of model coefficients

**3. Model Interpretation**  
- Assessed how accessibility and neighborhood convenience influenced property value
- Examined the role of transit proximity and local amenities in explaining price variation
- Used model results to translate statistical output into business insight

---

## Key Insight

Properties located closer to public transportation and in areas with more nearby convenience stores tended to have higher value per unit area. The project showed how pricing can be modeled using measurable location-based features.

---

## Business Value

- Demonstrated how regression can support real estate valuation decisions
- Identified key variables influencing property value
- Showed how location intelligence can improve price forecasting
- Reinforced the role of analytics in market-based decision-making
